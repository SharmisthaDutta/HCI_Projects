Recommended Guidelines

  1. Design process and evaluation
    • 1.1 Provide useful content to user
    • 1.2 Do not display unsolicited windows or graphics
    • 1.3 Organizing and placing the shared document
  2. How to identify who is in control of an object (which may or may not be visible)?
    • 2.1 Showing information about object owner
    • 2.2 Showing list of users currently viewing the shared object
  3. Passing control to another user and requesting control
    • 3.1 Widgets for request control
    • 3.2 Widgets for passing control
  4. Screen based controls for Multiuser App's Shared object
    • 4.1 Object Edit/update markers
    • 4.2 Have a widget for comment
    • 4.3 Display user editors but dont hide the shared content
    • 4.4 Placing of comment widget
    • 4.5 Display and Customize the Navigation Areas but do not hide the shared object
  5. Guidelines for colors:
    • 5.1 Colors that can and cannot be used
  6. Strategies for control change
    • 6.1 Ask/Request strategy
    • 6.2 RoundRobin, queue and other strategies
  7. Guidelines for effective content sharing
    • 7.1 Show Revision history of Shared object including timestamp and editor of shared object
    • 7.2 Give users flexibility with multiple platform User interfaces
