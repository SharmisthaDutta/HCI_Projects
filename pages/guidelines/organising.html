Guideline:

Place the object to be updated by multiple users in center position and to take at least half (50%)of
the User Interface(UI).

--------------------------------------------------------------------------------------------------------------------

Comments:

Provide a nice interface for the Users to locate and edit the Shared object.

-------------------------------------------------------------------------------------------------------------------

Sources:

The eight multiuser apps that we studied. Some apps were targeted towards a specific group of users e.g Software Developers and people working in a office set up. However there were several commonality between the apps that we can narrow down to make a good set of guidelines.

-------------------------------------------------------------------------------------------------------------------

Good Example:

Skype: It was very easy to locate shared object ( which was Voice in this case).

-------------------------------------------------------------------------------------------------------------------

Bad Example:

Dropbox: We were required to follow some instructions to set up the process for sharing an object, It was not very intuitive.
