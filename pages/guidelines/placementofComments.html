Guideline:

Place the comment widget(e.g icon) near the place of shared object source of comment and place the actual comment dialog box away
from the shared doc. When the user clicks the comment icon the comment dialog box should appear in a separate pane
so that everyone can see the comment and the shared object same time. comment box should not hide the shared object.

--------------------------------------------------------------------------------------------------------------------

Comments:

This guideline is applicable where shared object is visible. Comment box should not hide the shared object otherwise it will be difficult for the users to work on the comments ther
other users have given, leading to a disruptive and painful work.

-------------------------------------------------------------------------------------------------------------------

Sources:

Multi user apps that we studied.

-------------------------------------------------------------------------------------------------------------------

Good Example:

Google docs: very easy to work on the comments and very easy to find out the feature.

-------------------------------------------------------------------------------------------------------------------

Bad Example:

Dropbox: the comments appear right above the shared object content which hides parts of shared document.
