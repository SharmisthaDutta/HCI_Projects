  • 4.3 : Display user editors but don't hide the shared content

Guideline:

Provide a widget so that users can view who is editing a shared content but do not hide the shared content.

Comments:

Allow users to view who is editing a particular line of a shared object ( in case of a visible shared object multi user app) like Google docs but do not hide the shared content.

Sources:

Multi user apps that we studied.

Good Example:

Cloud9: is an excellent example. Here the lines that are edited by other user gets underlined with a different color and username appears in small box in such a way that it never hides the shared code.

Bad Example:

Microsoft office online, google docs: It hides the shared content in order to show the user editor of shared doc.
