Guideline:

Allow multiple users to edit the object by giving multiple cursors with different colors.Each user will get a different color cursor with their name appearing on top of the cursor..

Comments:

This guideline does not apply to the shared objects that are not visible aka the multi user voice apps do not need to follow these rule.

Sources:

Multi user apps that we studied.

Good Example:

Microsoft office online is a good example.

Bad Example:

Dropbox: does not provide any markers as such. It follows a check and check out protocol where if a user is making changes, the other user will view a lock on the item.
