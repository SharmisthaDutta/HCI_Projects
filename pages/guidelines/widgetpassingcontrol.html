Guideline:

Provide a pane to the presenter of shared object to pass control to another user to become presenter.

--------------------------------------------------------------------------------------------------------------------

Comments:

Provide a user list pane or some other widgets like buttons against each current user to organizers/presenters so that it enables them to make other users as organizers or presenters, grant or revoke attendee privileges and dismiss users .

-------------------------------------------------------------------------------------------------------------------

Sources:

Multi user apps that we studied.

-------------------------------------------------------------------------------------------------------------------

Good Example:

Go to meeting is an execellent example.

-------------------------------------------------------------------------------------------------------------------

Bad Example:

Google Docs: does not provide any choice as such as multiple users can simultaneously edit a document.
