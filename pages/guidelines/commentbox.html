Guideline:

Provide a widget to allow users to comment on the shared item. Widget can be a dialog box or a small button which upon click shows a comment window.

Comments:

Users have recommended that comment feature of a multi user app is a very essential feature as it helps in effective collaboration. User may want to comment on another user edited content in order to give suggestions and ask for suggestions.

Sources:

Multi user apps that we studied.

Good Example:

Google docs: Here users even have a chat window for comments where user can resolve an issue and reply in chat window.

Bad Example:

Cloud 9: it does not have a comment box explicitly but it has chat window pane where user can type in for collaboration.
