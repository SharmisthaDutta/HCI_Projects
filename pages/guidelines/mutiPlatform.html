Guideline:

Give users flexibility with multiple platform User interfaces, so that they can access the app anytime, anywhere

--------------------------------------------------------------------------------------------------------------------

Comments:

Users have commented that having the flexibility to access an multi user app from multiple platforms ( eg. Mobile, Desktop, Laptop, Tablet)
leads to a better functionality and productivity..

-------------------------------------------------------------------------------------------------------------------

Sources:

Multi users apps that we studied.

-------------------------------------------------------------------------------------------------------------------

Good Example:

Dropbox, Gotomeeting, Skype, Google Docs, Cisco webEx: provides a mobile User interface also. So we can access these apps anytime, anywhere.

-------------------------------------------------------------------------------------------------------------------

Bad Example:

Cloud9: We cannot access this IDE from mobile right now.
