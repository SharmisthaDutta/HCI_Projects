Guideline:

Provide a useful content to the users so that they can easily locate the shared object intuitively.

--------------------------------------------------------------------------------------------------------------------

Comments:

There is a common factor in most of the multi user app. That is the shared object. We can think of the shared object as main content in case of a website. Hence users should be intuitively identify and locate the shared obejct, without having to lost in details of finding it.

-------------------------------------------------------------------------------------------------------------------

Sources:

The eight multiuser apps that we studied. Some apps were targeted towards a specific group of users e.g Software Developers and people working in a office set up. However there were several commonality between the apps that we can narrow down to make a good set of guidelines.

-------------------------------------------------------------------------------------------------------------------

Good Example:

Google docs is an excellent example of useful content. This app is very intuitive in nature to be able to provide a engaging and relevant content .

-------------------------------------------------------------------------------------------------------------------

Bad Example:

Skype is a overall multi-user-friendly app but in shared content area they sometimes display Ads which distracts the user. A novice user might think Ads as part of shared content of another user.
