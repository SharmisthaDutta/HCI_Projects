Guideline:

Other application control strategies should include a organizer/presenter privileges where one user can grant or revoke attendee privileges and dismiss attendees, and be able to unmute or mute an attendee.

Comments:

Our research found that multi user app like Skype lacks the feature of organizer/presenter, so if there is a web conference call wtakes place where every one can speak, it would become very chaotic.

Sources:

Multi user apps that we studied.

Good Example:

Go to meeting is an excellent example of a smooth application control strategy

Bad Example:

Skype: It does not provide a organizer/presenter feature
