Guideline:

Provide a request access widget (a button on top of shared content) to control the shared object exclusively is a good feature .

--------------------------------------------------------------------------------------------------------------------

Comments:

Provide a request access widget to control the shared object exclusively is a good feature if
the app is used for heavy trafficked information. If there is only one controller who is
modifying the shared object (where applicable) then there are lesser chances of chaos and it would lead
to increased productivity.

-------------------------------------------------------------------------------------------------------------------

Sources:

Multi user apps that we studied.

-------------------------------------------------------------------------------------------------------------------

Good Example:

Cisco Webex, Skype for Business are excellent examples.

-------------------------------------------------------------------------------------------------------------------

Bad Example:

Google docs, Microsoft office online: any user of shared content can type in anytime they want to.
